Livox ROS Driver 2 is the second-generation driver package for connecting Livox LiDAR products to ROS and ROS2. It supports the HAP and Mid-360 LiDAR sensors and publishes point cloud data in multiple formats — including the standard `sensor_msgs/PointCloud2`, the Livox custom format, and the PCL `PointXYZI` format — as well as IMU data.

<Note>
  Livox ROS Driver 2 is intended for testing and development scenarios. For mass production deployments, you should optimize the driver source code to meet your specific requirements.
</Note>

## Supported hardware

Livox ROS Driver 2 supports the following LiDAR devices:

| Device | Description |
|--------|-------------|
| **HAP** | Solid-state LiDAR with high-density scanning |
| **Mid-360** | 360° hemispherical LiDAR for autonomous navigation |

## How it works

<Steps>
  <Step title="Install dependencies">
    Install ROS or ROS2, then build and install the Livox-SDK2 library.
  </Step>
  <Step title="Build the driver">
    Clone the repository into your ROS workspace and run the build script for your ROS version.
  </Step>
  <Step title="Configure your LiDAR">
    Edit the JSON config file to set your LiDAR's IP address, ports, and scan parameters.
  </Step>
  <Step title="Launch and visualize">
    Use one of the provided launch files to start the driver and visualize point clouds in RViz.
  </Step>
</Steps>
